# fleetctl

Homebrew formula for [fleetctl](https://fleetdm.com/guides/fleetctl) — the CLI tool for [Fleet](https://fleetdm.com) device management.

## Install

```sh
brew tap ds0x/fleetctl https://github.com/ds0x/fleetctl
brew install fleetctl
```

## Usage

```sh
fleetctl --help
```

For full documentation, see the [Fleet guides](https://fleetdm.com/guides/fleetctl).
